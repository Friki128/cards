Play

How to play:

There is one way to win, complete the board!! For this task, you will be given 3 cards at a time. Select one and place it!

Simple, isn't it? Wrong, because there is a caviat, you can only place cards in way that no opposing arrows touch each other. That means you cannot place this card anywhere around the previous card.

Accessability

You can always play using the tabulator and pressing enter to make a choice. You can identify cards by their code. The code indicates it's values and it's the name of the card. The code is four words, in clockwise order from the top, each word indicates a the color of that arrow. For example:

The code of this card would be blue-blue-red-red.
